# پروژه تحلیل دوپلر - قدم‌به‌قدم

یک پروژه آموزشی-پژوهشی برای تحلیل اثر دوپلر از مفاهیم پایه تا مباحث پیشرفته.

## 🎯 اهداف پروژه
- یادگیری عمیق اثر دوپلر
- تمرین برنامه‌نویسی علمی  
- مستند‌نویسی به زبان فارسی
- استفاده از Git و GitHub برای مدیریت پروژه

## 📁 ساختار پروژه

```bash
DopplerProject/
├── src/ # کدهای اصلی پایتون
├── docs/ # مستندات و تئوری
├── notebooks/ # آموزش تعاملی (Jupyter)
├── data/ # داده‌های نمونه
└── tests/ # تست‌های خودکار
```


## 🚀 شروع سریع
```bash
# کلون کردن ریپو
git clone https://github.com/adelgachkar/DopplerAnalysis-StepByStep.git

# رفتن به پوشه پروژه  
cd DopplerProject

# اجرای تست اولیه
python src/doppler_basic.py
```
